This repository contains all AMXX plugins I've ever made.

## w_glow.sma
Every weapon lying on the floor is now glowing, and the color depends on its ammo amount.

## w_skins.sma
Replaces default models (v_*.mdl, w_*.mdl, p_*.mdl) and allows you to choose through the menu. [Search for required models yourself.](https://gamebanana.com/skins/games/4254)

## w_pickup.sma
Hover the weapon, press E to pick-up.
